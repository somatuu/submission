# 집중력 관리 프로그램

## 프로젝트 정보
- 과목: 컴퓨팅사고와 SW 코딩
- 작성자: 하가연

## 프로젝트 설명
본 프로젝트는 사용자의 집중 시간을 기록하고,
집중 통계를 분석하며,
집중력 향상 또는 휴식을 위한 랜덤 행동을 추천하는
콘솔 기반 파이썬 프로그램이다.

본 프로그램은 기말 프로젝트에서 제공된
소프트웨어 요구사항 명세서(SRS)와
유스케이스 명세서를 바탕으로 구현되었다.

## 주요 기능
- 집중 세션 시간(분 단위) 기록
- 오늘 집중 통계 확인
  - 세션 개수
  - 총 집중 시간
  - 평균 집중 시간
  - 표준편차
- 전체 집중 기록 통계 확인
- 랜덤 집중/휴식 행동 추천
- 집중 기록 데이터 파일 저장 및 불러오기
